What are some existing tools that can be used for cryptographic inventorying?
**********************

The following tools are not an exhaustive list, but rather ones that are open source or ones that we have in our lab through our collaborators. Please refer to their websites and repositories to learn more about their capabilities.

Open source tools
--------------------------

*   `pqcscan <https://github.com/anvilsecure/pqcscan>`_ for scanning SSH and TLS servers

*   `sslscan2 <https://github.com/rbsec/sslscan>`_ to test SSL/TLS enabled services to discover supported cipher suites

*   `crt.sh <https://crt.sh/>`_ to find SSL/TLS certificates issued for specific domains or organizations

*   `cyberzero PQC Edge Scanner <https://scan.cyberzero.io/>`_ to scan for PQC transition signals at the public edge of any domain

NCCoE collaborator tools

*   `SandboxAQ AQtive Guard <https://www.aqtiveguard.com/>`_

*   `Data-Warehouse PCert <https://datawh.info/>`_ 

*   `Keyfactor AgileSec <https://www.keyfactor.com/products/cryptographic-discovery-inventory/>`_

*   `Tychon Cryptographic Inventory <https://tychon.io/cryptographic-inventory/>`_

*   `CodeQL <https://docs.github.com/en/code-security/concepts/code-scanning/codeql/about-code-scanning-with-codeql>`_ (also open source)
